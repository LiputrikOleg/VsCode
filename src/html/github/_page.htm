9. Основное меню

  • Code - Главная страница
  • Pull requests - Запросы на вытягивание

    Добро пожаловать в пул-реквесты! Запросы на включение помогают вам совместно работать над кодом с другими людьми. По мере создания запросов на включение они будут появляться здесь в списке с возможностью поиска и фильтрации. Для начала вам следует создать запрос на включение.

  • Discussions - Обсуждения

    Добро пожаловать в обсуждения! Обсуждения предназначены для обмена объявлениями, создания дискуссий в вашем сообществе, ответов на вопросы и многого другого. Для начала вы можете создать новое обсуждение.

  • Actions - Действия

    Начните работу с действиями GitHub Создавайте, тестируйте и развертывайте свой код. Сделайте проверки кода, управление ветвями и сортировку проблем так, как вы Начните работу с действиями GitHub Создавайте, тестируйте и развертывайте свой код. Сделайте проверки кода, управление ветвями и сортировку проблем так, как вы хотите. Выберите рабочий процесс, чтобы начать. Пропустите это и настройте рабочий процесс самостоятельно

  • Projects - Проекты

    Добро пожаловать в проекты Таблицы проекта, построенные как электронные таблицы, предоставляют вам живое полотно для фильтрации, сортировки и группировки проблем и запросов на извлечение. Настройте их в соответствии со своими потребностями с помощью настраиваемых полей и сохраненных представлений.

  • Wiki - Вики

    Добро пожаловать в каталог AT wiki! Вики-сайты предоставляют место в вашем репозитории, где можно вместе составить дорожную карту вашего проекта, показать текущий статус и лучше документировать программное обеспечение.
